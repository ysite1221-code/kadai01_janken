## ⑤工夫した点・こだわった点
- じゃんけんの結果次第で、自分の手・相手の手・勝敗・支払い金額の全てを変えるのが分岐が多くて整理が大変だった
- じゃんけんの結果をjqueryのdata属性で保持、その属性を参照することで会計・人数による自分の負担金額を計算するというプロセスを考えるのが大変だった
- ルールの表記については、小泉進次郎先生にInspireされた感じに仕上げました。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 視認性。機能の配置、画面の切り替え・表示タイミングなど、視認性のところがほぼ修正できず、ほぼ機能を羅列しただけ。
- 最初から、UXを意識してUIを考えておかないと、作りながらでは限界があるということがよくわかった。
- 数値を入力させる場合は、想定していない数値入力に対しての対応方法がWebにいくらでも転がっているので、そういうのも参考にしながら次回以降はもう少し実用性を考えて作っていきたい
